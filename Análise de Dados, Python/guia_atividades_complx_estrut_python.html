Bloco 2 — Revisão Objetiva de Complexidade (Simplificada)

ClasseIdeiaExemplo didático
O(1) Tempo constante, independe do tamanho. Acessar lista[i], dict[chave].
O(n) Proporcional ao tamanho n (um loop simples). Somar todos os elementos de uma lista com for.
O(n log n) Comum em ordenações eficientes. sorted(lista) em geral.
O(n²) Loops aninhados (comparar cada par). Verificar similaridade entre todos os elementos.

Observação: para esta etapa do curso, priorize raciocínios O(1) e O(n). Evite O(n²) quando possível.

Bloco 4 — Loops Simples (for / while)

Exemplos básicos com listas de números:

# Somar valores (for) valores = [10, 12, 15, 9] soma = 0 for v in valores: soma += v print("Soma:", soma) # Contar quantos valores estão acima de um limiar limiar = 10 contador = 0 for v in valores: if v > limiar: contador += 1 print("Acima do limiar:", contador) # while: consumir elementos de uma lista (ex.: buffer) buffer = [1, 2, 3] while buffer: x = buffer.pop() # remove do fim # processa x # ... pass

Bloco 5 — Funções (definir, reutilizar, testar)

Uma função isola uma tarefa. Mantenha nomes claros e parâmetros simples.

def media(valores): # Retorna a média aritmética de uma lista de números. if not valores: # lista vazia return 0.0 soma = 0 for v in valores: soma += v return soma / len(valores) def acima_do_limiar(valores, limiar): # Conta quantos valores são estritamente maiores que o limiar. cont = 0 for v in valores: if v > limiar: cont += 1 return cont # Testes simples print(media([1, 2, 3])) # 2.0 print(acima_do_limiar([1,2,3],1)) # 2

Bloco 7 — Exercícios (20 questões dissertativas, loops e funções)

  1. Listas e loop for: Dada uma lista de 10 números, escreva uma função soma_lista(lista) que retorne a soma. Explique por que a complexidade é O(n).
  2. Contagem com condição: Escreva uma função contar_acima(lista, limiar) que conte quantos valores são > limiar. Teste com três listas diferentes e explique.
  3. Média simples: Implemente media(lista) sem usar sum(). Explique o que acontece se a lista estiver vazia.
  4. Loop while: Use while para consumir elementos de uma lista (removendo do fim com pop()) e some os valores removidos. Explique quando usar while em vez de for.
  5. Função com dois retornos possíveis: Escreva min_max(lista) que retorne uma tupla (menor, maior). Explique a complexidade e como tratar lista vazia.
  6. Dicionário básico: Dado um dicionário cfg = {"sensorA": 30.0, "sensorB": 28.5}, escreva uma função que retorne o valor de uma chave com get() e default. Explique o custo médio O(1).
  7. Construindo dicionário: Dada uma lista de pares (sensor_id, valor), crie uma função que construa um dict {sensor_id: [valores...]} percorrendo a lista uma única vez.
  8. Leitura de CSV: Utilize Pandas para ler sensor_data_100k.csv (apenas 5.000 linhas). Liste as colunas disponíveis e explique como descobrir o tipo de cada coluna.
  9. Coluna numérica: Escolha uma coluna numérica (por exemplo, temperatura). Construa uma função simples que calcule média, min, max percorrendo uma lista de valores Python (sem NumPy) e aplique aos dados da coluna convertida para lista.
  10. Filtro simples: Escreva uma função que receba lista_valores e um limiar, e retorne nova lista com apenas os valores > limiar. Explique a complexidade.
  11. Contagem por sensor: A partir do DataFrame, crie um dicionário contagem_por_sensor que mapeia sensor_id → quantidade de registros usando apenas um loop for sobre df["sensor_id"].
  12. Média por sensor (básico): Construa uma função que calcule a média de uma coluna (por exemplo, temperature) por sensor_id usando dicionário de acumulação (somas e contagens). Evite groupby nesta etapa.
  13. Faixa de valores: Escreva uma função que conte quantos valores estão em uma faixa [a, b]. Aplique em uma coluna numérica e discuta a utilidade em IoT.
  14. Normalização simples: Implemente normaliza_0_1(lista) que transforma valores para o intervalo [0,1] usando min–max. Explique possíveis problemas com valores constantes.
  15. Janela deslizante (básica): Escreva uma função que, dada uma lista e um window, retorne a soma de cada janela contígua (sem Pandas). Discuta a complexidade.
  16. Detecção simples de picos: Dada uma lista, conte quantas posições i satisfazem lista[i] > lista[i-1] e lista[i] > lista[i+1]. Trate bordas.
  17. Função principal: Crie uma função main() que leia o CSV (5.000 linhas), escolha uma coluna numérica, calcule média/min/max, e imprima um pequeno relatório. Não use recursos avançados.
  18. Teste e validação: Explique como você testaria suas funções com dados pequenos artificiais (listas curtas) antes de usar o CSV.
  19. Tempo de execução: Meça o tempo de uma função simples (por exemplo, soma de lista) com time.perf_counter() para 10.000, 50.000 e 100.000 elementos. Explique por que o tempo cresce aproximadamente de forma linear.
  20. Reflexão: Comente quando você escolheria dicionário em vez de lista para procurar dados de um sensor específico. Dê um exemplo curto e justifique.
